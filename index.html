Steps to use :

  1. Click on Browse button to Select an Audio Recording in .MP3 format.
  2. Choose Play and Pause button to audio of selected Recording file.
  3. To Trim a Recording File :
    • On upload of Recording file A waveform will appear.
    • On the waveform at a point, Double click to select the Start-Time and drag till End-Point, where recording need to end.
    • Leave the click, to load the selected part on the table.
    • After new Recording is loaded you can able to Play, Download or Delete the mentioned Recording as per requirements.
    • Kindly Note: Download can take some time depending on file size, Your Patience is appreciated !!!
  4. Click Browse button to start from the beginning.
